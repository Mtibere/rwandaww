Our work

Know More

Rwanda We Want Organization has various areas of focus namely: Youth Empowerment, Youth Sexual Reproductive Health, Civic Education, Inclusion and to tackle all those areas, Rwanda We Want Organization has different program which are:.

IKIRARO

IKIRARO which translates to “bridge” is a program which as the name suggests bridges the youth to their ambitions using an experience-based approach by connecting them with successful, experienced fellow youth and a select few outstanding non-youth for guidance, inspiration, and motivation. This program is implemented through Weekly sessions and a BIRASHOBOKA conference.

RANGURURA

RANGURURA which translates to “raise your voice” is a program in which the youth sharpen their civic skills like public speaking and critical thinking. It is implemented through debates and public speaking competitions.

NDAKUZIRIKANA

NDAKUZIRIKANA which translates to “I care” is a program which promotes inclusion of disabled, marginalized and underprivileged youth in the community. It is implemented through advocacy campaigns, providing financial academic support and fundraisings.

UMURAGE

UMURAGE which translates to “inheritance” is a program in which the youth are taught about the history of their country that they are inheriting and generational knowledge transmission so as to learn from the past and be able to build a better future. It is implemented through conferences and seminars.
