Our Work!

Rwanda We Want Organization has various areas of focus namely: Youth Empowerment, Youth Sexual Reproductive Health, Civic Education, Inclusion and to tackle all those areas, Rwanda We Want Organization has different program which are:.

IKIRARO

IKIRARO program which as the name suggests bridges the youth to their ambitions using an experience-based approach by connecting them with successful, experienced fellow youth and a select few outstanding non-youth for guidance, inspiration, and motivation. This program is implemented through Weekly sessions and a BIRASHOBOKA conference.

1w ago Gethered 800 youth

RANGURURA

RANGURURA is a program in which the youth sharpen their civic skills like public speaking and critical thinking. It is implemented through debates and public speaking competitions.

1w ago 50 youth from 3 districts

NDAKUZIRIKANA

NDAKUZIRIKANA is a program which promotes inclusion of disabled, marginalized and underprivileged youth in the community. It is implemented through advocacy campaigns, providing financial academic support and fundraisings.

1w ago

UMURAGE

UMURAGE is a program in which the youth are taught about the history of their country that they are inheriting and generational knowledge transmission so as to learn from the past and be able to build a better future. It is implemented through conferences and seminars.

1w ago Gethered more than 1000 youth
